The Ultimate Student

The ultimate Student est un jeu de cartes qui fonctionne comme une gestion d'étudiant.
Nous avons à notre charge un étudiant qui est représenté par plusieurs stats :

  • La faim
  • La connaissance
  • L'argent
  • Le social

Plusieurs cartes vont défiler devant le joueur, ces cartes vont exercer une influence sur nos stats.

  • Les cartes Malus baissent forcément les stats.
  • Les cartes Bonus augment forcément les stats.
  • Les cartes événement vont influencer les stats suivant le choix.

Car oui ! Le joueur sera amené à choisir entre 2 propositions pour chaque carte devant lui. Ce qui va aussi pouvoir changer le type de stats à influencer.

VISUELS DU PROJET

Les cartes évènements

Ces cartes peuvent autant augmenter ou diminuer les stats.

Les cartes malus

Ces cartes vont automatiquement diminuer les stats, certaines contiennent un choix de malus.

Les cartes bonus

Ces cartes vont automatiquement augmenter les stats, certaines contiennent un choix de bonus.
